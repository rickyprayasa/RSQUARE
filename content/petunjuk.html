Panduan Penggunaan Template

Terima kasih! Ikuti langkah mudah di bawah ini untuk memulai.

1. Download File Panduan (PDF)

Klik tombol di bawah ini untuk mengunduh file panduan. Di dalamnya terdapat link untuk menyalin template Google Sheets Anda.

Memuat tombol download...

2. Buka PDF & Buat Salinan Template

Setelah mengklik link di dalam PDF, browser Anda akan membuka halaman dari Google Sheets. Klik tombol biru besar "Make a copy" untuk menyalin template ini ke dalam akun Google Drive Anda.

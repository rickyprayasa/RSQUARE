Panduan Penggunaan Template

Terima kasih telah mengunduh! Ikuti 3 langkah mudah di bawah ini untuk memulai.

1. Buka File PDF & Klik Link Akses

Buka file PDF panduan yang baru saja Anda download. Di dalam dokumen, Anda akan menemukan halaman berjudul "Akses Template nya disini". Klik link yang tersedia pada halaman tersebut untuk melanjutkan.

2. Buat Salinan Template (Make a copy)

Setelah mengklik link, browser Anda akan membuka tab baru yang menampilkan halaman dari Google Sheets. Klik tombol biru besar "Make a copy" untuk menyalin template ini ke dalam akun Google Drive Anda.

Penting: Proses ini aman dan hanya membuat salinan di akun Anda. File asli kami tetap tidak akan berubah.

3. Baca Panduan & Mulai Gunakan!

Selamat! Template sekarang menjadi milik Anda dan siap digunakan. Kami sangat menyarankan Anda untuk membaca keseluruhan isi PDF untuk memahami semua fitur canggih yang ada, seperti penggunaan Apps Script untuk menyimpan invoice.

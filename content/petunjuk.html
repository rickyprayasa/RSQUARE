Panduan Penggunaan Template

Terima kasih! Ikuti langkah mudah di bawah ini untuk memulai.

1. Download File Panduan (PDF)

Klik tombol di bawah ini untuk mengunduh file panduan. Di dalamnya terdapat link untuk menyalin template Google Sheets Kamu.

Memuat tombol download...

2. Buka File PDF & Klik Link Akses

Buka file PDF panduan yang baru saja Kamu download. Di dalam dokumen, Kamu akan menemukan halaman berjudul "Akses Template nya disini". Klik link yang tersedia pada halaman tersebut untuk melanjutkan.

3. Buat Salinan Template (Make a copy)

Setelah mengklik link, browser Kamu akan membuka tab baru yang menampilkan halaman dari Google Sheets. Klik tombol biru besar "Make a copy" untuk menyalin template ini ke dalam akun Google Drive Kamu.

Penting: Proses ini aman dan hanya membuat salinan di akun Kamu. File asli kami tetap tidak akan berubah.

4. Baca Panduan & Mulai Gunakan!

Selamat! Template sekarang menjadi milik Kamu dan siap digunakan. Kami sangat menyarankan Kamu untuk membaca keseluruhan isi PDF untuk memahami semua fitur canggih yang ada, seperti penggunaan Apps Script untuk menyimpan invoice.

Butuh Bantuan?

Jika Kamu mengalami kendala atau memiliki pertanyaan lebih lanjut, jangan ragu untuk menghubungi kami. Kami siap membantu Kamu!
